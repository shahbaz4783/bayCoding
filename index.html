About Me

Hello! My name is Shahbaz and I enjoy creating things that live on the internet. My interest in web development started back in 2020 when i explored so many stuffs on youtube in lockdown. That time i was more focussed on my school, so left it and this year (2022) started again with motive to became a full stack web developer.

Although my plan is to became a blockchain developer, web development is the first step in that direction. My main focus these days is learning things and implement those things and make it live on internet too.

I'm using Github Pages to make my created websites live on internet.

Here are a few technologies I've been learning these days:

  • HTML
  • CSS
  • Bootstrap
  • Javascript
  • Node.js
  • MongoDB
